Hymmnos Picture Generator  
===

An online Hymmnos picture generator. Use Python 2.7 & Django 1.11.4 & Pillow 4.2.1 at the bigenning.  
这是一个在线生成塔语字体图片的 Web 应用。最初创建时使用 Python 2.7 & Django 1.11.4 & Pillow 4.2.1.
